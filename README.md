## Overview

Audio on a modern Mac is rarely a single stream. Music plays in the background, a video call occupies the foreground, browser notifications interrupt at random intervals, and a podcast waits paused in another tab. <a href="#">Sound Control</a> was built specifically for this reality, providing the per-application volume management that macOS itself has never offered natively and that every multi-tasking user eventually finds themselves needing.

The application lives in the <a href="#">menu bar</a>, accessible with a single click at any moment without switching focus away from whatever is currently active. The interface presents every sound-producing application as its own channel with an independent volume fader and left-right balance control. Adjustments apply instantly and persist across sessions — an app that was set to 40% volume will still be at 40% the next time it produces sound, regardless of what the system volume is doing. This persistence is especially valuable for applications like communication tools, where a consistent and comfortable <a href="#">call volume</a> relative to other audio sources matters every time a meeting starts.

<a href="#">Audio device routing</a> per application solves a problem that has frustrated Mac users with multiple outputs for years. Rather than switching the system output device every time a different application needs a different destination, Sound Control allows each app to have its own assigned output — headphones for focused work applications, <a href="#">AirPlay speakers</a> for music, an audio interface for recording software. Changes happen without interrupting other apps and without navigating system preferences. The application also supports <a href="#">keyboard shortcuts</a> for volume control, assignable per application or globally, keeping audio management accessible without any mouse movement during work. Equalizer settings per application add a final layer of control, allowing the frequency response of individual apps to be shaped independently — boosting voice clarity in communication apps while leaving music playback untouched by the same adjustments, creating a genuinely <a href="#">professional audio workflow</a> on macOS that matches the flexibility found in dedicated studio environments.

## Key Features

- Per-application <a href="#">volume sliders</a> for independent audio level control
- <a href="#">Balance control</a> per app for left-right channel adjustment
- Custom <a href="#">audio device routing</a> assigning different outputs per application
- <a href="#">Menu bar</a> interface with instant access to all audio sources
- Assignable <a href="#">keyboard shortcuts</a> for per-app and master volume control
- <a href="#">Persistent settings</a> remembering volume levels across sessions
- Per-application <a href="#">equalizer</a> for frequency shaping per audio source
- <a href="#">AirPlay</a> and Bluetooth device support in per-app routing
- System-wide <a href="#">audio management</a> beyond macOS native volume controls

## Additional Information

Sound Control's menu bar design keeps audio management fast and unobtrusive. The full mixer is never more than one click away, and adjustments take effect immediately without any confirmation dialogs or settings windows standing between the user and the audio change they need.

The per-app equalizer brings a professional audio dimension to everyday Mac use. Shaping the frequency response of a video call application independently from music playback allows each audio source to be optimised for its specific content without compromise or conflict between sources.

Keyboard shortcut support completes Sound Control's role as a keyboard-first audio tool. Power users who keep their hands on the keyboard during work can adjust, mute, and route application audio without ever reaching for the mouse or trackpad.
